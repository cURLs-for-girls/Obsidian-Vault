## Glossar

| Begriff | Bedeutung |
|---|---|
| BWS | Brustwirbelsäule |
| Serratus | Serratus anterior – "Sägezahnmuskel" an der Seite des Brustkorbs |
| Kippung | Schulterkippung = Schulter nach hinten kippen (≠ Schulterblätter zusammendrücken) |
| Protraktion | Schulter nach vorne – das andere Ende vom Spektrum |
| Stapelung | Brustkorb über Hüfte, alles ausgerichtet |
| Triple Extension | Hüfte + Knie + Knöchel gleichzeitig gestreckt – Extension-Phase der Sprint-Mechanik |
| Druckmechanik | Werfen / Drücken nach vorne – für Menschen wichtiger als Zugmechanik |
| Drive | Zusätzliche Bewegungskomponente, die zur Hauptbewegung dazukommt – erzeugt mehr Spannung |
| Lateral Line | Seitliche Körperkette – wird mit Stöcken + Fußgewicht in Neigung trainiert |
| Innenrotation | Schulter/Arm dreht so, dass Daumen nach innen zeigt – Startphase Trizepsstreckung |
| Außenrotation | Gegenbewegung – kommt in der zweiten Phase mit Schulterkippung |
| Armdrive | Aktive Armbewegung beim Sprinten – „Schmeißen und Folgen" |
| FP | Functional Patterns |
