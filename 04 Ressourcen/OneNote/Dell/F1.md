# F1

Accuracy= TP+TN/TP+TN+FN+FP - wie genau das Modell ist --> alles was richtig ermittelt wurde

Precision= TP/TP+FP --> wie viele der pos. Vorhergewsagten wirklich positiv sind

Recall=TP/TP+FN --> von allen echten positiven, wie viele richtig ermittelt wurden

F1= 2* Precision * recall /precision + recall --> harmonisches mittelmaß (für raten) -> kleinere werte werden stärker bestraft
